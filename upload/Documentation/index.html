Ngôn ngữ Tiếng Việt cho Opencart 2.x

Giải nén tệp Vietnamese_language.zip

Upload thư mục catalog và admin trong thư mục (upload)lên thư mục cài đặt opencart của bạn.

Đăng nhập vào phần quản trị tìm menu ->System->Localisation->Language

Click Add Button

Điền các thiết lập như sau:

Bấm vào Save để hoàn tất

Thiết lập Tiếng Việt làm ngôn ngữ chính cho Opencart

Đăng nhập vào phần quản trị tìm menu ->System->Setting

Click Edit Button

Chọn Local Tab và Điền các thiết lập như sau:

Bấm vào Save để hoàn tất
